# 用（Practices）— 优秀实践沉淀

本目录按「产学研用」框架属于**用**：taiyi 在生产落地后沉淀的优秀实践。

## 用途

存放 Agent 真实使用中沉淀的可复用资产，例如：

- **Prompt 模板**：经过验证的、对特定场景效果好的 system prompt 片段
- **典型 Skill**：在生产中跑通过、可推广的技能（含 SKILL.md + quality_gate.md）
- **场景规则集**：特定业务域（如财务对账、客服退款）的规则包
- **运维经验**：部署、调参、故障排查的实战笔记
- **效果案例**：taiyi 实际解决的真实任务记录（脱敏后）

## 与其他部分的关系

- `learning/`（学）—— 为什么这么设计（理论）
- `research/`（研）—— 怎么验证的（测试 demo）
- 生产代码（`src/` 等）—— Agent 本体
- **`practices/`（用）—— 本目录，落地后的最佳实践**
